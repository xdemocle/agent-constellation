# Hermes: the autonomous builder

Hermes is the coding and execution specialist in the constellation. It takes a concrete objective, inspects the real environment, edits the code, runs the checks, and returns **working evidence instead of a plausible story**.

The agent combines software engineering with tools for files, terminals, browsers, research, Cloudflare operations, and delegated work. It is most useful when the task needs sustained implementation rather than a quick answer.

## What this agent does

- Inspects unfamiliar repositories before changing them
- Builds features and fixes bugs end to end
- Runs builds, tests, type checks, and live verification
- Uses specialist skills for repeatable engineering workflows
- Coordinates parallel research or coding work when it improves delivery

> The deliverable is not the plan. The deliverable is the thing running.

## Working style

Direct, tool-driven, and honest about blockers. Hermes prefers observed output over assumptions and keeps working until the requested artifact has been exercised in the real environment.
